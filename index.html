My projects

Project 1 - Personal web site

The very first attempt to apply the HTML & CSS knowledge to create a real project with a practical value.
I started with my personal web-site.
As I was coding from the scratch, it took me a while to finilize this project. The goal was also to use the maximum of my CSS knowledge and use as many features I could. Without "overloading" the final result of course.
Although a modern way of creating a responsive column layout, is to use CSS Flexbox, I wanted to practice an "old-school" CSS property of float as well. Important to mention that I am fully familiar with a Flexbox Layout module.
Color combination is inspired by my two favotite colors.
